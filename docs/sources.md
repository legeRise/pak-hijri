# Sources

Verified month-start records should include source information.

Acceptable source types include:

- Official Ruet-e-Hilal announcement
- Government or ministry announcement
- Trusted Pakistani news report
- Other reliable moon-sighting confirmation

Each source should ideally include:

- `source_name`
- `description`
- `url`
- `published_at`

Only `source_name` is required for a draft source. Add a link when one is available.
